Benefits of git and github

     . Great way to keep track of our database
     . A way to share and collaborate with others
     . Ability to save our process and go back to earlier points in our projects 
     Important git commands to push your code 
     step 1: Create a new repository
     step 2: Open the terminal
     step 3: Set your local project as the current working directory
     step 4: Initiatilize the local directory as a git Repository
     command: git init
     step 5: add the files in your new local repository. Your files will now be staged for their first commit.
     command: git add .
     command: git status
     step 6: Commit the files that you have staged in your local repository.
     command git commit -m "message"
     step 7: Copy the remote repository's URL.
     step 8: Add the URL for the remote repository where your local repository will be pushed.
     command: git remote add origin url
     step 9: Push the code in your local repository to Github.
     command: git push -u origin master
     step 10: View your files in your repository
     Tip: to change the drive in command line use D:
               
    

            
How to sort an array

     //Strings 
     let names = ["sahil","deepak","aashish","deepanshu"];
     names.sort();
     //output:---  ["aashish","deepak","deepanshu","sahil"]
     //Numbers
     let scores = [89,76,100];
     scores.sort((a,b) => {return a-b});
     //Output:--- [76,89,100]
     
     How to select a random element
     let items = ["apple","banana","mango"];
     let index = Math.floor(Math.random()*items.length);
     items[index]
     
     How to reverse a string
     function reverse(s){
     return s.split("").reverse().join("");
     }
     reverse("sahil");
     // Output:--- lihas
